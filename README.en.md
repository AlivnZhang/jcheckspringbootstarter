# jcheckspringbootstarter

#### Description
SpringBoot 参数校验框架
- `spring-boot-starter-web`: web应用，引入这个starter后，可以不用再引入`spring-boot-starter-web`
- `spring-boot-starter-aop`: 主要用到了AOP，面向切面编程。
- `aviator`: 谷歌的规则引擎，主要用来校验参数
- `fastjson`: 阿里Json工具
- `commons-lang3`: apache工具
- `lombok`: 工具
- `swagger2`: 生成接口文档
